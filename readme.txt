Extends WooCommerce with additional locations, such as UK counties, France departments, etc.

== Description ==

The plugin adds a number of additional location sets to WooCommerce. You can choose locations you want to be added. Check for the list of locations shipped with the plugin below.

If you have an idea about other locations to be included [let us know](http://wordpress.org/support/plugin/woo-locations/).


<ul>
    <li>UK, counties</li>
    <li>France, departments</li>
    <li>Austria, states</li>
    <li>Portugal, districts</li>
    <li>Ecuador, provinces</li>
    <li>Cyprus, cities</li>

</ul>
